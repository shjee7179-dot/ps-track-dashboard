# PS Track Dashboard

미래 의사과학자 챌린지 트랙 학습 여정 대시보드 프로젝트.

이 폴더는 기존 사업/기획 자료 폴더와 분리된 앱 프로젝트 루트로 사용한다. 이후 GitHub 저장소는 이 `ps-track-dashboard/` 폴더를 기준으로 생성한다.

## 현재 상태

- 기획 문서: [docs](./docs)
- 앱 코드: Sprint 0 foundation 구현 중
- 현재 브랜치: `codex/sprint-0-foundation`
- 주요 화면: `/login`, `/dashboard`, `/admin/users`, `/admin/roles`, `/admin/audit-logs`, `/admin/access-logs`, `/admin/pwa`, `/admin/settings`

## 개발 명령

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run build
```
