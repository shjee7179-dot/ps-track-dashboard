# MVP Scope

## 우선순위 정의

| 우선순위 | 의미 |
| --- | --- |
| P0 | 2026년 7월 운영 전에 반드시 필요 |
| P1 | 운영 품질을 크게 높이며 1차 중 붙일 기능 |
| P2 | 수동/CSV/외부 도구로 대체 가능하거나 2차 후보 |
| 제외 | 1차 MVP 범위 밖 |

12. 학생별 학습 여정
13. 학습피스 상태 관리
14. 산출물 목록
15. 산출물 제출 / 상세

## P2 화면

1. 기수 종료 보고서
2. 학생 성장 리포트
3. 멘토링 품질 리포트

5. 코드 / 분류값 관리
6. 다음 기수 복제 마법사

## 1차 제외

- 자체 화상회의
- 자체 채팅
- 자체 설문 빌더
- 자동 성과지표 산출
- 자동 리마인드 발송
- 고급 BI 대시보드
- 복잡한 다중 트랙 운영
- 네이티브 모바일 앱
- PWA 푸시 알림
- 오프라인 학습 기능
- 백그라운드 동기화

## 1차 포함 PWA 범위

- 모바일 반응형 웹
- PWA 기본 설정
- 홈 화면 추가
- 앱 아이콘 / manifest
- 로그인 세션 유지
- 주요 학생 화면 모바일 최적화
- PWA 설정 화면
