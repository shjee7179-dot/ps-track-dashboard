# Sprint 2 Task: 산출물과 멘토링

## 목표

개인/팀 산출물 제출, 멘토링 일정/기록, 산출물 피드백 흐름을 구현한다.

## 포함 범위

- 산출물 목록
- 산출물 제출 / 상세
- 개인 산출물
- 팀 산출물
- 산출물 리뷰
- 멘토링 일정
- 멘토링 기록
- 피드백 이력

## 주요 데이터

- Artifact
- Submission
- Feedback
- Team
- MentoringSession
- ActivityLog

## 완료 기준

- 산출물은 개인 또는 팀 소유로 생성된다.
- 산출물은 연결 학습피스와 학습성과를 가질 수 있다.
- 멘토링 기록은 대상 학생/팀, 연결 산출물, 후속 액션을 포함한다.
- 제출/수정/피드백 확인 로그가 남는다.

## 검증

- 학생은 본인 또는 소속 팀 산출물만 제출/조회한다.
- 멘토는 담당 학생/팀 산출물만 리뷰한다.
- 멘토링 불참 또는 미배정이 위험 신호로 잡힌다.
- 프로덕션 빌드가 통과한다.
