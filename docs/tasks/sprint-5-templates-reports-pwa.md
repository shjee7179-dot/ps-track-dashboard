# Sprint 5 Task: 템플릿 / 리포트 / 모바일 품질

## 목표

기수 반복 운영 기반, 외부 설문 링크, CSV 내보내기, 모바일/PWA 품질을 정리한다.

## 포함 범위

- 프로그램 템플릿
- 상대 일정 템플릿
- 템플릿에서 기수 생성
- 설문 링크 관리
- 설문 응답 여부
- 참여율 / 완료율 리포트
- 산출물 현황 리포트
- CSV 내보내기
- 모바일 최적화 점검
- PWA 품질 점검

## 주요 데이터

- ProgramTemplate
- ScheduleTemplate
- Survey
- LearningOutcome
- ReportExport
- PwaSetting

## 완료 기준

- 모듈, 콘텐츠, 학습피스, 루브릭, 학습성과, 상대 일정 템플릿을 다음 기수에 복제할 수 있다.
- 참여자/멘토 배정/제출물/피드백/평가 결과는 복제하지 않는다.
- 설문은 외부 링크와 응답 여부만 관리한다.
- 핵심 리포트를 CSV로 내보낼 수 있다.

## 검증

- 템플릿으로 생성한 기수에 실제 날짜가 계산된다.
- 복제 제외 데이터가 새 기수에 섞이지 않는다.
- 모바일 주요 화면에서 텍스트와 UI가 겹치지 않는다.
- PWA 설치 가능 상태가 확인된다.
- 프로덕션 빌드가 통과한다.
