# Sprint 1 Task: 학생 여정 중심 MVP

## 목표

학생 개인 기준의 학습 여정을 구현한다. 학생은 할 일을 보고, 운영자/멘토는 학생별 진행과 지연을 확인한다.

## 포함 범위

- 학생 대시보드
- 학생 상세
- 학습피스 목록 / 상세
- 학습피스 상태 관리
- 학생별 학습 여정
- 콘텐츠 열람 핵심 로그

## 주요 데이터

- User
- Cohort
- Module
- Content
- LearningPiece
- ActivityLog
- RiskSignal

## 완료 기준

- 학습피스 유형별 완료 기준을 표시한다.
- 학생 대시보드는 이번 주 할 일과 마감 임박 항목을 우선 표시한다.
- 운영자/멘토는 scope 내 학생의 여정을 볼 수 있다.
- 학습피스 상태 변경이 ActivityLog에 기록된다.

## 검증

- 학생 계정은 본인 학습피스만 볼 수 있다.
- 모바일 폭에서 학생 대시보드가 깨지지 않는다.
- 지연 학습피스가 위험 신호 후보로 잡힌다.
- 프로덕션 빌드가 통과한다.
