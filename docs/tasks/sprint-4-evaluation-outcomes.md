# Sprint 4 Task: 평가와 성과

## 목표

산출물 유형별 루브릭 평가와 학습성과 매핑을 구현하고 PI 대시보드의 품질/성과 요약을 제공한다.

## 포함 범위

- 루브릭 평가
- 루브릭 항목 관리
- 학습성과 목록 / 상세
- 학습성과 매핑
- 학생별 성과 증거
- PI 대시보드

## 주요 데이터

- LearningOutcome
- Rubric
- RubricItem
- Evaluation
- Artifact
- Feedback

## 완료 기준

- 산출물 유형별 루브릭을 적용할 수 있다.
- 루브릭 항목은 학습성과와 연결될 수 있다.
- 여러 평가자의 평가를 지원한다.
- PI는 산출물 품질과 성과 요약을 볼 수 있다.

## 검증

- 평가자는 권한 범위 내 산출물만 평가한다.
- 항목별 점수와 종합 코멘트가 저장된다.
- 평가 결과가 산출물 상세와 PI 대시보드에 반영된다.
- 프로덕션 빌드가 통과한다.
