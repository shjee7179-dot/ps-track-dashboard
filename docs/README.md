## 현재 합의된 핵심 결정

- 학생 개인 기준 학습 여정을 기본 축으로 둔다.
- 학습피스 완료 기준은 유형별로 다르게 둔다.
- 산출물은 개인/팀 모두 가능하고 여러 개를 허용한다.
- 멘토링은 일정, 기록, 산출물 피드백, 루브릭 평가까지 연결한다.
- 권한은 역할 + scope로 설계한다.
- 2026년 1차는 트랙 1개, 50명 이하 운영에 최적화한다.
- 감사 로그, 접속 로그, PWA 설정, 전체 시스템 설정은 1차 P0에 포함한다.
- 자체 설문 빌더, 자체 채팅, 자체 화상회의, 자동 리마인드 발송은 1차에서 제외한다.
